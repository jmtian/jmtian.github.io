Emacs 命令笔记

Posted on 2022-09-29

Emacs 命令笔记

快捷键

  • C-x h

    全选

  • C-u C-x =

    查看光标当前字符信息(字符编码,位置信息,字体等)

M-x 执行命令

  • whitespace-mode、global-whitespace-mode

    显示控制字符

  • untabify

    将tab替换成空格, 先选中文本

  • profiler-start、profiler-report

    查看性能（ cpu/mem/cpu+mem）

  • 去除行尾^M

    M-x replace-string RET, C-q C-m RET, RET

配置命令

  • (setq default-tab-width 4)

    设置tab宽度

  • (setq c-basic-offset 4)

    设置tab宽度

  • (setq c-default-style "linux")

    设置代码风格

org-mode （The Org Manual）

  • #+OPTIONS: tags:nil

    设置不导出tags

  • #+OPTIONS: toc:nil

    设置不导出Table of contents

  • #+OPTIONS: toc:n

    设置Table of contents 导出等级

  • #+OPTIONS: tex:{}

    设置tex转换模式, 被{}包含

  • #+OPTIONS: num:nil

    Table of contents 前不显示序号

  • #+TITLE: this is a title

    设置标题

  • #+STARTUP: inlineimages

    启动时显示内嵌图片文件

  • C-c C-e

    将org导出到其它格式

  • C-c C-c

    操作, 不同元素有不同行为

  • Shift-right/left

    为标题设置TODO、STARTED、DONE等

  • Tab, Shift-Tab

    隐藏、显示标题内容

  • Alt-right/left

    修改当前标题等级

  • Alt-up/down

    上下调整标题顺序

  • C-c C-d

    设置 dead line

  • - [ ]

    [X] , 设置为taggle, 可使用 C-c C-c 选中或取消选中

  • [/] 和 [%]

    [1/3] 和 [33%] , 设置为显示子项目进度比例和百分比, 使用 C-c C-c 初始化数据

  • C-c [

    File move to front of agenda file list

  • M-x org-agenda

    Enter agenda views

  • <s TAB

    add source code block, C-c ' edit source code, C-c C-c at source code, will process code.
